
# Tools

> A development toolkit running in a browsers

## Supported functions

* 压缩图片
* 域名/IP
* POST/GET 模拟请求
* JSON 格式化
* 时间戳转换
* 二维码 生成/识别
* URL 解码/编码
* 文本翻译
* BASE64编码


## Develop

``` bash
# git clone https://github.com/vatezj/tools.git
# cd tools
# yarn install && yarn run dev
```
> At this point, the browser opens a window [localhost:8080](http://127.0.0.1:8080)

## Build
``` bash
# npm run build
```
